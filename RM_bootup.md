## 1. What Renaissance Ministries Is (30-second summary)

Renaissance Ministries is a Christian formation project centered on Sunday fellowship discussions and the development of Kingdom culture. The **Christos AI** family of modules provides AI-assisted Christian formation, including: theological training (Seminar), Bible study (Cross-Check), interfaith engagement (Council), political citizenship (Voting Network), cultural engagement (Commons), and specialized modules for medical, counseling, and other domains.

The core insight: **God is everything** — there is nothing that exists apart from God's sustaining consciousness. Evil is not a second principle but the rejection of God's nature by free creatures. This resolves the problem of evil and provides a framework for engaging intellectual objections to Christianity.

## 11. Conventions

### Essay Naming
- Fellowship essays: `Fellowship_Discussion_[Topic]_[Month][Year].md`
- Newsletter articles: `Newsletter_[Title]_[Month][Year].md`
- Specifications: `Christos_[Module]_Technical_Specification_v1.html`
- Operating Systems: `Christos_[Module]_Operating_System_v1.md`

### Version Control
- Grammar: `v1.X` — increment minor version after each update
- Specifications: `v1` — increment major version for significant changes
- Essays: No versions — date in filename

### Scripture References
- King James Version preferred for quotations
- Book Chapter:Verse format (e.g., John 3:16)

### Formatting
- Markdown for text documents
- HTML for technical specifications (with CSS styling)
- PDF for final distribution
